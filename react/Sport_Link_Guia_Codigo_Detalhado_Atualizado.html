Navegação

Visão geral

O app tem quatro partes principais.

📰

Notícias

NewsAPI + Football-Data para notícias, jogos e fases.

🎯

Bolão

Login, grupos, palpites e ranking no Supabase.

📍

Jogar

Mapa, peladas e criação de partidas.

🎮

Joguinho

IA, online, som, ranking e Canvas.

Fluxos principais

Navegação

  1. App.js abre NavigationContainer
  2. Tab.Navigator monta as abas
  3. Cada aba importa uma tela

Atualização

  1. RefreshControl chama onRefresh
  2. Funções fetch buscam APIs
  3. Estados atualizam listas

Football-Data

  1. Chamada com X-Auth-Token
  2. Recebe JSON de matches/standings
  3. Filtra status, stage ou data

Joguinho Online

  1. Host cria sala
  2. Guest entra por código
  3. Supabase guarda estado
  4. WebView recebe estado

noticias.js + APIs esportivas

A tela Notícias integra NewsAPI e Football-Data.

fetchNoticias(query)
Busca notícias na NewsAPI.
q
Texto pesquisado nos artigos.
language=pt
Retorna notícias em português.
sortBy=publishedAt
Ordena por mais recentes.
pageSize=6
Limita quantidade de artigos.
fetchDadosFutebol
Busca dados esportivos.
X-Auth-Token
Header da Football-Data.
/competitions/WC/matches
Endpoint de jogos da Copa.
/competitions/WC/standings
Endpoint de classificação.
item.homeTeam / item.awayTeam
Times da partida.
item.score.fullTime
Placar final.
item.stage
Fase do torneio.
RefreshControl
Atualiza puxando a tela.
WebView
Abre notícia dentro do app.
NewsAPI /v2/everything: use q para busca, language=pt para português, sortBy=publishedAt para recentes e pageSize para limitar quantidade.
Busca na NewsAPI
const res = await fetch(
  `https://newsapi.org/v2/everything?q=${encodeURIComponent(query)}&language=pt&sortBy=publishedAt&pageSize=6&apiKey=${NEWS_API_KEY}`
);
Football-Data: use o header X-Auth-Token. A Copa é WC. /matches retorna partidas e /standings retorna classificação.
Busca de jogos
const resMatches = await fetch(
  'https://api.football-data.org/v4/competitions/WC/matches?status=SCHEDULED,IN_PLAY,PAUSED,FINISHED',
  { headers: { 'X-Auth-Token': FOOTBALL_DATA_TOKEN } }
);
